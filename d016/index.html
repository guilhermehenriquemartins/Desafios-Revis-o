CURSO HTML e CSS

Esse curso é introdutório, que vai desde o Módulo 01 até o 05, após a conclusão de todos os módulos o aluno estará capacitado à criar sites em nível intermediário. Espero que goste do curso e, acima de tudo, treine bastante, e em seguida, assista, e depois treine bastante novamente. Pois é com treino e repetição que os melhores foram forjados!

CURSO JAVASCRIPT

Esse curso é para quem está iniciando no aprendizado da linguagem de programação JavaScript, sendo bastante utilizada para a construção de sites, normalmente desenvolvido por dev front-end, porém hoje é bastante comum também usá-lo para programação back-end. Apesar disso, esse curso se trata exclusivamente para quem deseja iniciar e se especializar na parte front-end do desenvolvimento.
